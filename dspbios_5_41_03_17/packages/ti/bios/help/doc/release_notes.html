DSP/BIOS Help

 Product v5.3x Release Notes

Last updated: 01 Oct 2007

This release note covers general product level issues and is divided into four sections, "General Information", “Limitations”, "Fixed In This Release", and "Validation Info".     

General Information

DSP/BIOS Help is a set of Microsoft Windows help files that contain help topics on DSP/BIOS. These help files are integrated into Code Composer Studio when a DSP/BIOS is selected via the CCS Component Manager DSP/BIOS Selector.

Limitations

In CCS 3.3, the DSP/BIOS 5.3x Tutorial is not available from CCS Help->Tutorial menu.   User's must select Help->Content, then look for DSP/BIOS 5.3x help topic, the tutorial is a sub topic under this heading.

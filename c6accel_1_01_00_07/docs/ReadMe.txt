ReadMe

This folder contains the documentation for the C6accel package
The documents contain 
* C6Accel Reference guide: 
  This reference guide provides an overview of C6accel and contains installation and user guide.
* Signal Processing Kernels in C6accel
   This document describes the C6accel`s Signal processing kernel wrapper API calls 
* Math Kernels in C6Accel 
   This document describes the C6accel Math kernel wrapper API calls
* Image processing kernels in C6accel
   This document describes the C6accel Image processing kernel wrapper API calls
* Advanced Features of C6Accel 
   THis document describes design of C6Accel, chaining C6Accel kernel API calls, Integrating C6accel in user codec server and how a    user can add their own algorithm to C6Accel .  

Note: The documentation corresponds to the latest release version of C6Accel [release 1.01.00.06] so may not be in sync with the older releases. 
The most recent documentation can be viewed on http://processors.wiki.ti.com/index.php/C6Accel:_ARM_access_to_DSP_software_on_TI_SoCs
The advantage of looking at Wiki:
•	Easy readability
•	There may be corrections of modifications for improving the documentation

    33    /*!
    34     *  ======== package.xdc ========
    35     *  Codec Engine encode-decode example which utilizes image codecs that
    36     *  implement the 1.0 xDM image interfaces.
    37     *
    38     *  This example builds using the XDC build tools, and is portable to
    39     *  several execution environments.  The following files are
    40     *  included in this package:
    41     *  @p(dlist)
    42     *      - `package.xdc`
    43     *          This file is required to declare a globally unique name for this
    44     *          package.
    45     *      - `package.bld`
    46     *          This file is the build script describing the application to the
    47     *          XDC built tools.
    48     *      - `makefile`
    49     *          A usability makefile wrapper around the XDC build tools.
    50     *      - `local.cfg`
    51     *          The XDC configuration file for this executable, when configured
    52     *          to run "local" codecs.  This is the typical configuration for
    53     *          single processor excution environments.
    54     *      - `local.tcf`
    55     *          The BIOS configuration file for this executable.  This file is
    56     *          only necessary when building for a BIOS runtime environment.
    57     *          It's base name must match that of the XDC config file.
    58     *      - `remote.cfg`
    59     *          The XDC configuration file for this executable, when configured
    60     *          to run "remote" codecs.  This is the typical configuration for
    61     *          multiple processor excution environments.
    62     *      - `app.c`
    63     *          The primary source file for this executable.
    64     *      - `main_BIOS.c` and `main_native.c`
    65     *          The execution runtime-specific entry point for this executable.
    66     *          This isolates any runtime-specific startup code from the generic
    67     *          application.
    68     */

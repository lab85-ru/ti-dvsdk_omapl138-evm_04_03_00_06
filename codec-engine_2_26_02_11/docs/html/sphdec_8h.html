sphdec.h File Reference

Detailed Description

The SPHDEC speech decoder codec interface. Provides the user an interface to create and interact with XDAIS algorithms that are compliant with the XDM ISPHDEC speech decoder interface.

#include <ti/xdais/dm/xdm.h>

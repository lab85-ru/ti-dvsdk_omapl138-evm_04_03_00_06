DVSDK Encode demo Overview

Purpose:

DVSDK encode demo shows how to encode video and audio or speech using algorithms and the Codec Engine Engine from Texas Instruments on the DM368 EVM.

The DM368 ARM core runs the demos on the Linux operating system, and all peripherals are controlled through Linux device drivers. The ARM also displays a user interface on the OSD (On Screen Display) and takes input from a USB mouse.

Output: Demo output will be displayed on your HD displayed connected to the EVM component video output port.

Input: Demo will capture the input from the sensor module connected on main board.

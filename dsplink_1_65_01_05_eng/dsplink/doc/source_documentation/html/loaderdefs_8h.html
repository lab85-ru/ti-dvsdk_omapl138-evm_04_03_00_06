BINLOADER_ImageInfo

Structure defining information about the image to be loaded. A pointer to this structure is passed during the PROC_load () function as the imagePath, when the binary loader is used.

============================================================================

Parameters:
gppAddrAddress of the file to be loaded, in GPP address space.
dspLoadAddrDSP address where the binary file is to be loaded.
dspRunAddrDSP address from where the binary file execution is to be started.
fileSizeSize of the file to be loaded in bytes.
shmBaseAddrAddress of the symbol where the DSPLINK shared memory base address is stored.
See also:

xDAIS 5.00 Release Notes

This release note is divided into the following sections: General Info, What's New, Documentation.

General Information

This xDAIS 5.00 release consists of the following packages:

  • ti.xdais - xDAIS interface and documentation
  • ti.xdais.dm - xDAIS-DM (xDM) interface and documentation

What's New

This is the xDAIS 5.00 release. It only contains interfaces and documentation, not the traditional xDAIS tooling.

Changes From Beta 2

  • Documentation updates to xDM classes.

Changes from Beta 1

  • Documentation updates to audio and video classes.

Resolved Defects

Documentation

The following documentation is available in this release:

  • xDAIS documentation list
  • xDM Users Guide
  • xDM API Reference Guide

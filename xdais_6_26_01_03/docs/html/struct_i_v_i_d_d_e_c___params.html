Detailed Description

Defines the creation time parameters for all IVIDDEC instance objects.

Remarks:
The application should set the parameters to 0 to use the algorithm's default values.
This structure may be extended by individual codec implementation allowing customization with vendor specific parameters.
The size field must be correctly set by the caller.

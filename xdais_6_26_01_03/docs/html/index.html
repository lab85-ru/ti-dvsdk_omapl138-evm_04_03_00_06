XDAIS Interface Reference

dais-t03

Modules

The XDAIS interfaces are comprised of the following:

  • XDAIS Types and Constants
  • IALG - XDAIS Algorithm Interface
  • IDMA
  • IDMA2
  • IRES
  • IDMA3 (C64P) - (EDMA3 only)

The XDM interfaces included in this release are comprised of the following:

  • General definitions
    • Shared XDM Definitions
  • Video
    • XDM Video Interface
    • IVIDDEC2 - XDM IVIDDEC2 Video Decoder Interface
    • IVIDENC1 - XDM IVIDENC1 Video Encoder Interface
  • Image
    • IIMGDEC1 - XDM IIMGDEC1 Image Decoder Interface
    • IIMGENC1 - XDM IIMGENC1 Image Encoder Interface
  • Speech
    • ISPHDEC1 - XDM ISPHDEC1 Speech/Voice Decoder Interface
    • ISPHENC1 - XDM ISPHENC1 Speech/Voice Encoder Interface
    • Speech interfaces have also been seperated into those which are common across all variants of speech codecs, and those which apply to a particular class:
      • XDM Speech/Voice Interface
      • XDM Speech/Voice Interface (AMR)
      • XDM Speech/Voice Interface (EVRC)
      • XDM Speech/Voice Interface (G.723)
      • XDM Speech/Voice Interface (G.726)
      • XDM Speech/Voice Interface (PCM)
      • XDM Speech/Voice Interface (SMV)
      • XDM Speech/Voice Interface (WBAMR)
  • Audio
    • XDM Audio Interface
    • IAUDDEC1 - XDM IAUDDEC1 Audio Decoder Interface
    • IAUDENC1 - XDM IAUDENC1 Audio Encoder Interface
  • Video Transcoding
    • IVIDTRANSCODE - XDM IVIDTRANSCODE Video Transcoder Interface

These older XDM interfaces continue to be supported, but also continue to be deprecated in this release. These interfaces are comprised of the following:

  • Video
    • IVIDDEC1 - XDM IVIDDEC1 Video Decoder Interface
    • IVIDDEC - XDM IVIDDEC Video Decoder Interface
    • IVIDENC - XDM IVIDENC Video Encoder Interface
  • Imaging
    • IIMGDEC - XDM IIMGDEC Image Decoder Interface
    • IIMGENC - XDM IIMGENC Image Encoder Interface
  • Speech
    • XDM Speech/Voice Interface
    • ISPHDEC - XDM ISPHDEC Speech/Voice Decoder Interface
    • ISPHENC - XDM ISPHENC Speech/Voice Encoder Interface
  • Audio
    • IAUDDEC - XDM IAUDDEC Audio Decoder Interface
    • IAUDENC - XDM IAUDENC Audio Encoder Interface

And finally, there are a few beta interfaces provided with this release. They are available for review, and comments are welcome. Each beta interface will be reviewed and updated with customer feedback, and when appropriate, upgraded to a supported interface in a future release.

  • IVIDANALYTICS - XDM Beta Video Analytics Interface
  • IUNIVERSAL - XDM Beta Universal Algorithm Interface
